Sentiment Analysis Tool

A web-based Sentiment Analysis Tool that classifies text into Positive, Negative, or Neutral sentiment using Hugging Faceâ€™s NLP models. The app provides detailed score breakdowns and visualizations with interactive pie charts and a history sidebar.

Features

Real-time text input and sentiment analysis.

Voice input support.

Visualize sentiment scores with interactive pie charts.

Maintain history of analyses in a sidebar.

Click on past analyses to view detailed modal with pie chart and sentiment.

Responsive UI with smooth sidebar and modal transitions.

Technology Stack

Frontend: React, Recharts, React Icons

Backend: FastAPI

Database: SQLite

NLP Model: Hugging Face twitter-roberta-base-sentiment

HTTP Client: Axios for frontend-backend communication

Usage

Enter text in the input field and hit Analyze (or press Enter).

View sentiment result and pie chart of score breakdown.

Access history in the sidebar to review past analyses.

Click a past analysis to open a modal with the detailed result.

Use the microphone button to input text via voice.
